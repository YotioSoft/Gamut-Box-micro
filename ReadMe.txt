					Gamut Box micro 0.1

----------------------------------------------------------------------------------------------------

1.はじめに
　この度は、Gamut Box microのα版をダウンロードしていただきありがとうございます。
　これはアルファ版ですので動作保証は出来ません。ご了承ください。

2.操作方法
　2.1 タスクバーとウィンドウ操作、ファイルの開き方
　　　「Gamut Box micro」とタイトルが書かれた黒い部分でマウスをクリックして動かすとウィンドウを動かうことが出来ます。
　　　
　　　次に、タスクバー上にある4つのボタンについて、左からご説明します。
　　　
　　　　ファイルを開く・・・「...」と書かれたボタン。ここからmp3ファイルなどの再生ファイル、リストファイル（.gbl形式）、インターネットショートカット（.url形式）を開きます。
　　　　最前面表示　　・・・画鋲のような絵柄が書かれたボタン。押すと最前面表示になります。
　　　　最小化ボタン　・・・「_」と書かれたボタン。ウィンドウを最小化にします。
　　　　終了ボタン　　・・・「×」と書かれたボタン。Gamut Box microを終了します。
　
　2.2 再生操作
　　　ウィンドウ右側に表示されている3つのボタンで再生・一時停止・停止を操作します。
　　　再生を「停止」すると、ファイル自体が閉じますのでご注意ください。
　　　
　　　また、リスト再生時には、その下にトラック操作ボタンが表示されます。

3.リスト再生について
　前作のGamut Boxに引き続き、リスト再生に対応しています。
　リスト再生には.gbl形式のリストファイルが必要です。
　gbl形式のリストファイルは、gamutbox_list.exeで作成できます。
　
　リストファイルは、再生ファイルと同じように「...」ボタンから開いてください。

4.歌詞表示について
　前作のGamut Boxに引き続き、lrc形式のファイルにて歌詞の表示に対応しています。
　ただし、現在のところGamut Box microには歌詞ファイルの作成機能は搭載しておりません。
　前作のGamut Boxの歌詞ファイル作成機能をご利用になるか、
　あるいは別のlrc形式の歌詞ファイル作成ソフトをご利用ください。
　lrcファイルは必ずlyricsフォルダに保存してください。

5.インターネットからの再生
　Gamut Box microでは、新しくインターネットショートカットを利用したインターネットからの再生に対応いています。
　YouTubeやインターネットラジオなどの音声を再生できます。
　この際、再生を操作することは出来ませんのでご注意ください。
